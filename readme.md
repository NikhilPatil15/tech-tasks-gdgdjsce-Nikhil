## 🧩 Folder Descriptions

### **1️⃣ Task-1-Figma-Blueprint**

This folder contains the implementation for **GDG DJSCE – Tech Task 1 (Figma Design Replication)**.

**Purpose:**
To replicate a given **Figma design** precisely using **Next.js** and **Tailwind CSS**, ensuring pixel-perfect alignment and full responsiveness.

**Key Highlights:**

- Built using **Next.js**, **TypeScript**, and **Tailwind CSS**
- Fully **responsive** on mobile, tablet, and desktop
- Contains pages:

  - 🏠 Home Page (`Home.png`)
  - 🚗 Vehicles Page (`Vehicles.png`)
  - 📄 Car Details Page (`Details.png`)

- Includes configuration files and assets required for running the Next.js project
- Documentation: [`Task-1-Figma-Blueprint/readme.md`](./Task-1-Figma-Blueprint/task-1-figma-blueprint/readme.md)
